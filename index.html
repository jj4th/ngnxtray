ngnxtray

A system tray application providing simple actions related to nginx installations on Windows. Requires .Net 3.5.

Installation

Place in the directory where you installed nginx, and double click to execute. Click the system tray icon to start, stop, or restart nginx. Editing the conf/nginx.conf file is also supported. Notepad.exe is used as the conf file editor.
